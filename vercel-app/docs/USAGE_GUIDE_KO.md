# 재고 사용 입력 가이드 (직원용)

> 모바일 앱에서 재고 사용을 입력·확정하는 방법을 안내합니다.

---

## 1. 화면 구성

**사용** 탭을 누르면 다음 순서로 표시됩니다.

1. **품목 목록** – 사용할 재고 목록 (카테고리별)
2. **분수·수량 영역** – 수량 선택 및 입력
3. **사용 품목** – 추가한 품목 목록 (장바구니)
4. **사용 확정** 버튼

---

## 2. 기본 사용 순서

### ① 품목 선택

- 아래 품목 목록에서 사용한 재고를 터치합니다.
- 선택된 품목은 파란색으로 표시됩니다.

### ② 수량 설정

- **분수 버튼** 또는 **+ / - 버튼**으로 수량을 맞춥니다.
- 필요하면 **수량 입력칸**에 직접 숫자를 입력할 수도 있습니다.

### ③ 사용 추가

- **[사용 추가]** 버튼을 눌러 장바구니에 넣습니다.

### ④ 반복 및 확정

- 다른 품목도 같은 방식으로 추가합니다.
- 모두 입력한 후 **[사용 확정]** 버튼을 눌러 확정합니다.

---

## 3. 분수 버튼 설명

### 첫 번째 줄 (큰 수량)

| 버튼 | 의미    | 수량 예시 |
|------|---------|-----------|
| 1    | 1개     | 1         |
| ½    | 2분의 1 | 0.5       |
| ¼    | 4분의 1 | 0.25      |
| ⅕    | 5분의 1 | 0.2       |
| ⅙    | 6분의 1 | 약 0.167  |
| ⅒    | 10분의 1| 0.1       |
| 1/25 | 25분의 1| 0.04      |
| ⇄    | **전환** – 두 번째 줄로 전환 |

### 두 번째 줄 (작은 수량)

| 버튼   | 의미     | 수량 예시   |
|--------|----------|-------------|
| 1/50   | 50분의 1 | 0.02        |
| 1/100  | 100분의 1| 0.01        |
| 1/200  | 200분의 1| 0.005       |
| 1/1200 | 1200분의 1| 약 0.00083 |
| ⇄     | **전환** – 첫 번째 줄로 돌아가기 |

---

## 4. + / - 버튼

- 선택한 **분수 크기만큼** 수량이 증가·감소합니다.
- 예시  
  - ½ 선택 후 + 누름 → 0.5씩 증가  
  - ⅒ 선택 후 + 누름 → 0.1씩 증가  
  - 1/100 선택 후 + 누름 → 0.01씩 증가  

---

## 5. 작은 수량 입력 (1/50, 1/100, 1/200, 1/1200)

작은 단위는 **수량 입력칸**과 **적용** 버튼을 사용하면 편합니다.

### 사용 방법

1. **⇄** 버튼으로 두 번째 줄로 전환
2. 사용할 분수 버튼 클릭 (예: 1/100)
3. **수량 입력칸**에 배수 입력 (예: 5)
4. **[적용]** 버튼 클릭 → 수량에 반영

### 예시

| 분수  | 입력한 수 | 결과 수량 |
|-------|----------|-----------|
| 1/100 | 5        | 0.05      |
| 1/50  | 10       | 0.2       |
| 1/200 | 4        | 0.02      |
| 1/1200| 3        | 0.0025    |

---

## 6. 사용 예시

### 예시 1: 설탕 0.5kg 사용

1. 설탕 품목 선택  
2. ½ 버튼 클릭  
3. [사용 추가] 클릭  

### 예시 2: 소금 0.1kg 사용

1. 소금 품목 선택  
2. ⅒ 버튼 클릭  
3. [사용 추가] 클릭  

### 예시 3: 식용유 0.05L 사용 (1/100 × 5)

1. 식용유 품목 선택  
2. ⇄로 두 번째 줄 전환  
3. 1/100 버튼 클릭  
4. 수량 칸에 **5** 입력  
5. [적용] 클릭  
6. [사용 추가] 클릭  

### 예시 4: 양이 애매할 때

1. 가장 가까운 분수 버튼 클릭  
2. + / - 버튼으로 맞춤  

---

## 7. 주의사항

- 품목을 반드시 먼저 선택한 후 수량을 설정하세요.
- 사용한 재고를 모두 추가한 뒤 **[사용 확정]**을 눌러야 재고에 반영됩니다.
- 확정 전에는 장바구니에서 **🗑️** 버튼으로 품목을 삭제할 수 있습니다.
